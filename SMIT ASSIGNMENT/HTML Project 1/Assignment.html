HTML is a standardised system for tagging text files that creates the structure for just about every page that we find and use on the web. It’s HTML that adds in page breaks, paragraphs, bold lettering, italics, and more. HTML works to build this structure by using tags that tell browsers what to do with text. For example, to make a word appear bold, we put that word between the following tags bold. The first tag indicates the start of the word(s) that we want to bold, and the closing tag(/) indicates where we want the bold to stop. It’s the basis for pretty much every page on the web.
